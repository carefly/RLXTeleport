# RLX Warp

A mod for warp to a specific location

This mod is based on LeviLamina

RLX Server Utility Plugin

## Install

lip install github.com/carefly/RLXWarp

## Usage

### for player:

- **/warp ls**
  
  list all warp

- **/warp <warp_name>**
  
  warp to a warp

---

### for op:

- **/warpop add**
  
  add a warp where you are

- **/warpop del <warp_name>**
  
  delete a warp

- **/warpop reload**
  
  reload the warp file
